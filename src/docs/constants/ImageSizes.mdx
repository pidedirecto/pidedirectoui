# ScreenSizes

You can import it from `import { ImageSizes } from '@pidedirecto/ui/constants'`

```ts
export const ImageSizes = Object.freeze({
    EXTRA_SMALL: 'EXTRA_SMALL',
    SMALL: 'SMALL',
    MEDIUM: 'MEDIUM',
    LARGE: 'LARGE',
    EXTRA_LARGE: 'EXTRA_LARGE',
});
```

- `EXTRA_SMALL` 40x40 image
- `SMALL` 80x80 image
- `MEDIUM` 120x120 image
- `LARGE` 160x160 image
- `EXTRA_LARGE` 200x200 image
