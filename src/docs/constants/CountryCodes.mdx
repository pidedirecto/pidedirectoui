# CountryCodes

You can import it from `import { CountryCodes } from '@pidedirecto/ui/constants'`

```ts
export const CountryCodes = {
    MX: 'MX',
    SE: 'SE',
    US: 'US',
    UY: 'UY',
    AR: 'AR',
    BR: 'BR',
    PE: 'PE',
    CL: 'CL',
    PY: 'PY',
    CO: 'CO',
};
```
