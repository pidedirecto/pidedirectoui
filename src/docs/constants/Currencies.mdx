# Currencies

You can import it from `import { Currencies } from '@pidedirecto/ui/constants'`

```ts
export const Currencies = {
    MXN: 'MXN',
    SEK: 'SEK',
    USD: 'USD',
    UYU: 'UYU',
    ARS: 'ARS',
    BRL: 'BRL',
    PEN: 'PEN',
    CLP: 'CLP',
    PYG: 'PYG',
};
```
