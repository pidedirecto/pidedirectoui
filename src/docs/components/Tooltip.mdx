# Tooltip
Renders a tooltip.

<Canvas of={TooltipStories.Primary} />

If you don't pass any children the tooltip will just render a question mark icon indicating that there is a tooltip there, but if you pass any children the tooltip will be wrapping the content passed and will be shown when you hover the content.

<Canvas of={TooltipStories.Children} />
