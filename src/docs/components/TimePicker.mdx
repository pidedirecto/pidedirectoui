# TimePicker
Renders a time picker.

<Canvas of={TimePickerStories.Primary} />

The `onChange` prop will be called, as first argument you will receive a date object with the hour and minute will be the selected hour,
and as a second argument you will receive the hours string with this format hh:mm

```tsx
import { TimePicker } from '@pidedirecto/ui'

const handleDate = (date: Date, hours: string) => {
    // Handle date
}

<TimePicker
    value={value}
    label={'Select a date'}
    name='input'
    onChange={handleDate}
/>

```
