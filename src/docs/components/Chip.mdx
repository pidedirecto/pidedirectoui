## Clickable

The `Chip` component can be clickable if you pass an `onClick` callback, besides this the component accepts any React.Node valid content as a children

<Canvas of={ChipStories.Clickable} sourceState='none' />

```tsx
import { Chip } from '@pidedirecto/ui';
import { EditIcon } from '@pidedirecto/ui/icons';

<Chip onClick={() => alert('You click on the chip!!')}>
    <div>
        Edit
        <EditIcon size={10} />
    </div>
</Chip>;
```
