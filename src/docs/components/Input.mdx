# Input

Renders an input

<Canvas of={InputStories.Primary} />

You can control the input value using a React state

```tsx
import { Input } from '@pidedirecto/ui'

const [value, setValue] = useState('')


const handleInput = value => setValue(value)


return <Input name='input' value={value} onChange={handleInput} />
```

If you want the HTML onChange event you can receive it as second parameter in the onChange callback prop.

If you need to override the default input maybe because you have an special format component like displaying currencies but keeping the same style you can pass the `InputComponent` prop

```tsx
import { Input } from '@pidedirecto/ui';

function MainComponent() {
    return <Input name='input' InputComponent={MyInput} label='Write your favorite food' />;
}

function MyInput({ ...props }) {
    return <SpecialInput {...props} />;
}
```
