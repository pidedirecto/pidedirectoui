# ConfirmDialog
Renders an ConfirmDialog.

The ConfirmDialog has several ways of use, for which there is a variant with the aim of letting the user understand if they are sure to perform an action.

```tsx
 const confirmDialog = useConfirmDialog()

const handleClick = async () => {
    const confirm = await confirmDialog({
                title:'Error Dialog title',
                content:'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean tempor ultricies ipsum. Fusce turpis sem, facilisis id dignissim et, dignissim commodo turpis. Morbi massa libero',
                acceptButtonText:'Accept',
                cancelButtonText:'Cancel',
                variant:'warning'
            })
        }

return (
    <div>
        <ConfirmDialog/>
        <Button onClick={handleClick}>Click me to show confirmDialogState!</Button>
    </div>
);
```
