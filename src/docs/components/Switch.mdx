# Switch

Renders a switch

<Canvas of={SwitchStories.Primary} />

If you need to control the switch state you can pass a boolean in the checked prop
```tsx
const [checked, setChecked] = useState(false)

const handleSwitch = () => {
    setChecked(!checked)
}

<Switch
  label="Are you a cheese lover?"
  name="switch"
  value="value"
  checked={checked}
  onChange={handleSwitch}
/>
```
