# DraggableList

Renders a draggable list

<Canvas of={DraggableListStories.Primary} sourceState={'none'} />

```tsx
import { DraggableList, DraggableListItem } from '@pidedirecto/ui';

<DraggableList onDragEnd={(values) => alert('The sorted values are ' + values.join(', '))}>
    <DraggableListItem value={'1'}>Do the laundry</DraggableListItem>
    <DraggableListItem value={'2'}>Wash the dishes</DraggableListItem>
    <DraggableListItem value={'3'}>Clean my room</DraggableListItem>
</DraggableList>;
```

To use the draggable list you have to use it with the `DraggableListItem`, this component needs a value prop, it can be any type and this value will be returned sorted in the `onDragEnd` callback from the `DraggableList`.
The `DraggableListItem` can has any children, not only plain string so the list item can be as customizable as you want.

The `DraggableList` can has any children too, but we don't recommend to wrap the `DraggableListItem` into some other elements because could lead to unexpected behaviours.
This does not mean that you can't move the list items to its own components, you can, but just try to avoid wrap them into other container elements.

**Do**

```tsx
import { DraggableList, DraggableListItem } from '@pidedirecto/ui';

<DraggableList onDragEnd={(values) => alert('The sorted values are ' + values.join(', '))}>
    <Component1 />
    <Component2 />
    <Component3 />
</DraggableList>;

function Component1() {
    return <DraggableListItem value={'1'}>Do the laundry</DraggableListItem>;
}

function Component2() {
    return <DraggableListItem value={'2'}>Wash the dishes</DraggableListItem>;
}

function Component3() {
    return <DraggableListItem value={'3'}>Clean my room</DraggableListItem>;
}
```

**Don't**

```tsx
<DraggableList onDragEnd={(values) => alert('The sorted values are ' + values.join(', '))}>
    <Component1 />
    <Component2 />
    <Component3 />
</DraggableList>;

function Component1() {
    return (
        <div className={'someClass'}>
            <div className={'container'}>
                <DraggableListItem value={'1'}>Do the laundry</DraggableListItem>
            </div>
        </div>
    );
}

function Component2() {
    return (
        <div className={'someClass'}>
            <div className={'container'}>
                <DraggableListItem value={'2'}>Wash the dishes</DraggableListItem>
            </div>
        </div>
    );
}

function Component3() {
    return (
        <div className={'someClass'}>
            <div className={'container'}>
                <DraggableListItem value={'3'}>Clean my room</DraggableListItem>
            </div>
        </div>
    );
}
```
