# Text

You can import it from `import { Text } from '@pidedirecto/ui'`

<Canvas of={TextStories.Large} />

## Variant

The `Text` component has some predefined styles if you want to render heading element o regular texts,
this can be done through the variant prop

- Title
<Canvas of={TextStories.Title} />

- Subtitle
<Canvas of={TextStories.Subtitle} />

- Paragraph
<Canvas of={TextStories.Paragraph} />

These variants also change the HTML element being rendered, for heading variants it renders the respective `h1` or `h2` heading element,
for the paragraph variant it renders a `p` element, by default, this component renders a `span` element.
