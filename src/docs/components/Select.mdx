# Select

Renders a select

<Canvas of={SelectStories.Primary} />

The Select component needs an array of options, each option is an object containing a `label` and `value` key, the label can be any string, the value can be either a string or a null value, make sure the option values are unique.
The option with null value should be used when the user is able to select an "empty" option.

To control the component you must pass the value prop

```tsx
import { Select } from '@pidedirecto/ui'

const [value, setValue] = useState(undefined)

const handleSelect = selectedValue => setValue(selectedValue)

return (
    <Select
        value={value}
        placeholder='Select an option'
        optioins={[]}
        onChange={handleSelect}
    />
)

```

The select is an implementation of the DropDown component

## Custom options

The `options` array requires the fields `label` and `value`, the label field will be used to render the option, but if you want more customizable content in the option, you can pass a `content` field,
this field accepts any `React.Node` value, but the `label` will be still required and will be used to show what option has been select.
