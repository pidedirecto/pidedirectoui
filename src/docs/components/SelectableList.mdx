# SelectableList

Renders a SelectableList

<Canvas of={SelectableListStories.Primary} sourceState='none' />

```tsx

import { SelectableList } from '@pidedirecto/ui'

const [value, setValue] = useState()

return (
    <SelectableList options={options} onChange={setValue} value={value} label={'Select an option'} />
)
```

The options array should be an array of objects containing a `value` and `label` keys, the `value` can be any type but should be unique between other options,
the `label` key must be string type, you can also pass a `content` key, it can be any React.Node type, and will be the content rendered next to the checkbox in the list,
even if you pass a content key the `label` field is required.

## Multiselectable

The `SelectableList` component can be multiselectable if you pass the `multiselectable` prop

## Virtualized

The `SelectableList` component can be virtualized, this is helpful if you have a large data set, you have to pass `virtualized` prop and `height` prop,
the `height` prop is required when virtualizing the list.
