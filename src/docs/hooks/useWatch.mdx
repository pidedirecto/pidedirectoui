# useWatch

Form hook to get value of inputs linked by name.
This hook is just an implementation of `useWatch` from `react-hook-form` so for more details see [docs](https://react-hook-form.com/docs/usewatch)

You can import it like

```ts
import { useWatch } from '@pidedirecto/ui/hooks';
```
