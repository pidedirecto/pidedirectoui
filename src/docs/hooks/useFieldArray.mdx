# useFormContext

This hook is just an implementation of `useFieldArray` from `react-hook-form` so for more details see [docs](https://react-hook-form.com/docs/usefieldarray)

You can import it like
```ts
import { useFieldArray } from '@pidedirecto/ui/hooks'

```
