# useForm

form hook to initialize a form component.
This hook is just an implementation of `useForm` from `react-hook-form` so for more details see [docs](https://react-hook-form.com/docs/useform)

You can import it like
```ts
import { useForm } from '@pidedirecto/ui/hooks'
```
