# useConfirmDialog

`useConfirmDialog` hook returns a function that you can call to show a confirmation dialog.

The basic usage for using this hook is first importing the ConfirmDialog component and put it somewhere in your applications,
commonly in the application main component

```tsx
import { ConfirmDialog } from '@pidedirecto/ui'

return (
    <ConfirmDialog />
)
```

We strongly recommend importing and using only one `ConfirmDialog` component per application, having 2 instances of this component
could lead to unexpected behaviours.

Once you have imported the `ConfirmDialog` component, you can use the `useConfirmDialog` hook anywhere in your application and start use confirmation dialogs .

```tsx
import { Button } from '@pidedirecto/ui';
import { useConfirmDialog } from '@pidedirecto/ui/hooks';

function Component() {
    const confirmDialog = useConfirmDialog();

    const handleClick = async () => {
        const confirm = await confirmDialog({
            title: 'Error Dialog title',
            content:
                'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean tempor ultricies ipsum. Fusce turpis sem, facilisis id dignissim et, dignissim commodo turpis. Morbi massa libero',
            acceptButtonText: 'Accept',
            cancelButtonText: 'Cancel',
            variant: 'warning',
        });
    };

    return (
        <div>
            <Button onClick={handleClick}>Click me to show confirmDialogState!</Button>
        </div>
    );
}
```
