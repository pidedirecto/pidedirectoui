# useFormContext

This hook is just an implementation of `useFormContext` from `react-hook-form` so for more details see [docs](https://react-hook-form.com/docs/useformcontext)

You can import it like
```ts
import { useFormContext } from '@pidedirecto/ui/hooks'

```
