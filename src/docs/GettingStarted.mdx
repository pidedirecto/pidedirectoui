To install the Directo UI library run the next command

```
npm i @pidedirecto/ui
```

## Usage

General components can be imported from the root of the package

```ts
import { Button, Table, Tabs } from '@pidedirecto/ui';
```

To import form components they should be imported from the `form` path

```ts
import { Form, FormCheckbox, FormTextField } from '@pidedirecto/ui/form';
```

To import hooks they should be imported from the `hooks` path

```ts
import { useNotification, useTheme } from '@pidedirecto/ui/hooks';
```

To import icons they should be imported from the `icons` path

```ts
import { BagIcon, BillIcon } from '@pidedirecto/ui/icons';
```

To import constants they should be imported from the `constants` path

```ts
import { ImageSizes, ScreenSizes } from '@pidedirecto/ui/constants';
```

To import utils they should be imported from the `utils` path

```ts
import { logStore } from '@pidedirecto/ui/utils';
```
