# PickersProvider

`PickersProvider` component allows you config the `DatePicker`, `DateTimePicker` and `TimePicker` and all other components using these components.
This is required to fully integrate these components with the app

```tsx
import { PickersProvider } from '@pidedirecto/ui';
import moment from 'moment'
import { theme } from 'src/theme/AppTheme';

<PickersProvider momentInstance={moment} muiTheme={theme}>
    <App />
</PickersProvider>
```

This only receives two props, `momentInstance` receives the moment instance of the projects, this is needed for the translations of the days, month, etc.
`muiTheme` is for styling the picker colors, fonts and more styles properties.
