# Theme

Component to override default components theme.

The `Theme` component receives the new theme as prop

```ts
type theme = {
        palette?: {
            primary?: {
                color?: string;
                border?: string;
            },
        };
        font?: {
            size?: {
                normal?: string;
                small?: string;
            },
            color?: {
                normal?: string;
                light?: string;
            }
        }
    }
```

If you use this component the `theme` prop is required but the values inside the theme object are optionals, so you can customize the theme as little as you want

```tsx
import { Theme } from '@pidedirecto/ui'

return (
    <Theme
        theme={{
            palette: {
                primary: {
                    color: 'red'
                }
            }
        }}
    />
)
```

You can only have one theme per application, so if you use multiple `Theme` components the last one will override the previous theme values
